# 設計ドキュメント: ダッシュボード画面の全面刷新

## 1. 目的
現在のダッシュボードは情報の密度が高く、カード形式のレイアウトにより視覚的な断片化が発生している。また、予実管理において固定費と変動費が混在しており、日割り予算などの行動指針となる情報が不足している。これらを解消し、直感的かつ洗練されたデザインへと刷新する。

## 2. デザインコンセプト: "Vibrant Minimalist"
- **脱カードレイアウト**: 境界線（Border）や影（Shadow）を排し、余白（White Space）とタイポグラフィを活かしたドキュメント・フロー形式を採用する。
- **鮮やかな配色**: インディゴ（メイン）、エメラルド（成功/プラス）、アンバー（注意/警告）、ローズ（超過/マイナス）を効果的に使用し、情報の重要度を直感的に伝える。
- **日本語化**: 全てのラベルと見出しを日本語で統一し、親しみやすさを向上させる。

## 3. 画面構成と機能要件

### 3.1 ヘッダー (Header)
- **左上**: ロゴ（Kakeibo-ai）を配置。
- **右上**: 月選択コンポーネント（◀ 2026年 6月 ▶）を配置し、過去データの閲覧を容易にする。

### 3.2 AIアドバイス・エリア
- ページ最上部に配置し、現在の家計状況に対するAIからのフィードバックを提示する。
- 視認性を高めるため、専用のバッジ（AIアドバイス）を付与する。

### 3.3 KPIハイライト (Main Stats)
- 行動指針となる3つの重要指標を横並びで表示する。
  1. **今日使える額**: `(変動費の月間予算 - 実績) / 残り日数` で計算される自由枠の目安。
  2. **総資産**: 現在の総資産額と前月比の増減。
  3. **精算待ち**: 立替金の未精算合計額と件数。

### 3.4 予実管理セクション (Budget Management)
- **タイムフレーム切替**: セクション上部にタブ（日次、週次、月次、四半期、年次）を配置。
- **変動費の予実**: 
  - プログレスバー内に「ペースメーカー（オレンジの縦線）」を表示し、現在の日数経過に対する理想ラインを可視化。
  - 数値表記: `￥実績 (±差分) / ￥予算`（例: `￥18,000 (-￥2,000) / ￥30,000`）。差分は理想ペースとの乖離を示す。
- **固定費の状況**: 
  - 変動費とレイアウトを統一するが、ペースメーカーや差分計算は表示しない。

### 3.5 資産分析セクション (Asset Analysis)
- **資産推移**: 折れ線グラフで資産の増減トレンドを表示。
- **資産内訳**: 円グラフ（ドーナツチャート）で現金・投資・ポイントなどの構成比を表示。

### 3.6 最近の支出 (Recent Activity)
- 直近の取引明細を、ドキュメント・フローに馴染むシンプルなリスト形式で表示。

## 4. 技術設計
- **Frontend**: React + TypeScript + Tailwind CSS。
- **Components**: 
  - `ProgressBar`: ペースメーカーおよび差分表示機能を追加。
  - `StatGroup`: カードを使わないフラットな統計表示。
  - `MonthSelector`: コンテキストに応じた月の切り替え。
- **State Management**: React `useState` または `useMemo` による期間別の動的な予実計算。

## 5. 成功基準
- ダッシュボードを開いた際、3秒以内に「今日あといくら使えるか」が把握できること。
- 前月データの閲覧が2クリック以内で完結すること。
- 固定費と変動費の状況が視覚的に明確に分離されていること。
