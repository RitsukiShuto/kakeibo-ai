# Issue: 手動インポート機能と初期設定UIの実装 (Issue #110)

## 概要
CSVファイルからの手動インポート機能と、マネーフォワード連携（`mf_session`）などの初期設定をUIから行えるようにします。これにより、コマンドラインを介さずにシステムの導入と運用が可能になります。

## 背景
- マネーフォワード連携に必要なクッキー設定などを手動で `settings.json` に書き込むのはユーザー体験として改善の余地がある。
- 自動連携していない銀行やカードのデータをCSVでインポートするニーズがある。
- Phase 2 のタスクとして予定されていたが、利便性向上のため早期に設計・実装を検討する。

## 完了定義 (Definition of Done)
- [ ] ブラウザからCSVファイルをアップロードし、DBへインポートできる。
- [ ] 設定画面からマネーフォワードのセッション情報や、AIキャラクターの設定などを保存できる。
- [ ] 保存された設定に基づき、その後の自動取得（fetch）が正常に動作する。
- [ ] `qa-tester` によるUI経由のインポート・設定変更テストをパスする。
