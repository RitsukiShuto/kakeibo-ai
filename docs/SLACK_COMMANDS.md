# Slack スラッシュコマンド一覧

Kakeibo AI Review System では、Slack から直接家計状況の確認や分析の実行が可能です。

## コマンド一覧

| コマンド | 引数 | 説明 |
| :--- | :--- | :--- |
| **`/kakeibo-review`** | `daily`, `weekly`, `monthly`, `skip` | **家計簿の分析を実行します。** [分析モード] [データ取得スキップ設定] を指定可能です。 |
| **`/kakeibo-check`** | なし | **今月の残予算と進捗率を即座に表示します。** 予算合計、実績、残額を表示します。💰 |
| **`/kakeibo-model`** | `pro`, `flash`, `lite` | **分析に使用するモデルを変更します。** 引数なしで現在の設定を確認できます。 |
| **`/kakeibo-stats-ai`** | なし | **今月の合計トークン使用量を表示します。** AI の利用統計を確認できます。🤖 |
| **`/kakeibo-last`** | なし | **最新の分析レポートを再表示します。** 直近のレポート内容を再掲します。📝 |
| **`/kakeibo-help`** | なし | **ヘルプを参照します。** 利用可能なコマンドの一覧を表示します。 |

---

## 互換用コマンド (`/review`)

基本形式: `/review [期間] [skip]`

| コマンド | オプション | 説明 |
| :--- | :--- | :--- |
| **`/review`** | `d`, `w`, `m`, `y`, `q` | `/kakeibo-review` と同等です。 |

---

## 使用例

- **今月あといくら使えるか知りたい時**:
  `/kakeibo-check`
- **今日使ったお金をすぐに分析してほしい時**:
  `/kakeibo-review daily`
- **より高精度なモデル（Pro）に切り替えたい時**:
  `/kakeibo-model pro`
- **今月の AI 課金状況を確認したい時**:
  `/kakeibo-stats-ai`
