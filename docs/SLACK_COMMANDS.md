# Slack スラッシュコマンド一覧

Kakeibo AI Review System では、Slack から直接家計状況の確認や分析の実行が可能です。

## 統合コマンド (`/kakeibo`)

基本形式: `/kakeibo [アクション] [オプション]`

| アクション | オプション | 説明 |
| :--- | :--- | :--- |
| **`check`** | なし | **今月の予算状況を確認。** 予算合計、実績、残額、および進捗率（%）を即座に表示します。💰 |
| **`last`** | なし | **最新の分析結果を再表示。** 過去の通知が流れてしまった場合に、直近のレポートを再掲します。📝 |
| **`stats`** | `ai` | **AI 使用量（トークン数）の表示。** 今月の累計プロンプト・レスポンス・合計トークン数を表示します。🤖 |
| **`review`** | `daily`, `weekly`, `monthly`, `yearly` | **手動分析の実行。** 指定した期間の家計分析をその場で開始します。引数なしの場合は `weekly` です。 |
| **`character`** | `gal`, `butler`, `zen` | **AI キャラクターの変更。** 分析レポートの口調や性格を切り替えます。引数なしで現在の設定を確認できます。👗 |
| **`help`** | なし | このヘルプガイドを表示します。 |

---

## 互換用コマンド (`/review`)

基本形式: `/review [期間] [skip]`

| コマンド | オプション | 説明 |
| :--- | :--- | :--- |
| **`/review`** | `d`, `w`, `m`, `y`, `q` | `/kakeibo review` と同等です。 |
| **`skip`** | - | データ取得（MoneyForward等からのフェッチ）をスキップして、既存のDBデータのみで分析します。 |

---

## 使用例

- **今月あといくら使えるか知りたい時**:
  `/kakeibo check`
- **今日使ったお金をすぐに分析してほしい時**:
  `/kakeibo review daily`
- **執事にお金の使い方をたしなめてほしい時**:
  `/kakeibo character butler`
- **今月の AI 課金（トークン）が不安な時**:
  `/kakeibo stats ai`
