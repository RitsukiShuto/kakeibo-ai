# 今回の分析ミッション：日次速報 (Daily Quick Scan)

## 📅 集計基準の明示
- **集計期間**: 今月1日から「今日」までのデータを対象としています。
- **集計基準日**: 入力データに含まれる `### 分析期間` および `#### 今回追加された差分明細` の最新日付を確認し、**「いつ（何日の夜/何時時点）の分析か」をレポートの最初に必ず明記してください。**
  - 例: 「✨ 5月20日時点の家計簿レビューだよ！」

## 🔬 チェックポイント
1. **今日の収支状況**: 今日発生した収入と支出を対比し、収支バランス（プラスかマイナスか）をフィードバックして。
2. **今日の店舗レビュー**: お金を使った場所（店名）から、今日の買い物の質をギャル視点で評価。
3. **正確な日割り予算の算出**: 
   - 今月の残り予算から**「1日あたりに使える額（日割り）」を計算する際、固定費（家賃、光熱費、通信費、サブスク等）は除外して計算してください。**
   - 固定費は毎月必ず発生するものであり、日割り計算に含めると「実際に自由に使えるお金」が正確に把握できなくなります。
   - 計算式: `(今月の総予算 - 固定費合計 - 今月既に使った変動費) ÷ 今月の残り日数`
   - この「1日あたりの自由予算」が、ユーザーが今日から月末までに毎日使える目安の金額です。
4. **明日のミッション**: 明日、即実行できる具体的な節約・改善行動を一つ提案して。
