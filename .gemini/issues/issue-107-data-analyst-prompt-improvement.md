# Issue: Data Analyst エージェントのプロンプト改善と実用性評価 (Issue #107)

## 概要
`@data-analyst` エージェントの定義（`.gemini/agents/data-analyst.md`）および、データ分析に使用されるプロンプトテンプレート（`prompts/system_prompt.md` 等）の見直しと改善を行います。また、実際のデータを用いた分析結果の質を評価し、使い勝手を向上させます。

## 背景
- 現在のプロンプトでも動作はしているが、より深い洞察（インサイト）の抽出や、ユーザーの行動変容を促す具体的なアドバイスの生成能力を高める余地がある。
- AIのキャラクター（ギャル風など）と分析の専門性のバランスを最適化したい。
- プロンプトテンプレートが複数のファイルに分散している可能性があり、保守性を高める必要がある。

## 完了定義 (Definition of Done)
- [ ] 現状のプロンプトの課題（具体性の欠如、誤認、トーンの不一致など）が特定されている。
- [ ] `@data-analyst` のエージェント定義が、最新のプロジェクト方針に合わせて更新されている。
- [ ] `prompts/` 配下のテンプレートが整理・改善され、分析精度が向上している。
- [ ] 改善後のプロンプトを使用した分析結果のサンプルが、実用面で合格点（ユーザーへの有用性、具体性、親しみやすさ）に達している。
- [ ] `qa-tester` による（出力形式などの）バリデーションをパスする。
