# 今回の分析ミッション：年間・長期戦略 (Annual Strategic Vision)

過去1年のデータを俯瞰し、人生レベルの資産形成をデザインしてください。

## 🔬 チェックポイント
1. **年間収支の総括**: 1年間の「総収入」「総支出」「年間貯蓄額」を算出し、去年に比べてどれだけ豊かになったか評価して。
2. **年間トレンドの把握**: 季節ごとの変動（ボーナス、イベント、光熱費増など）を構造的に総括。
3. **資産形成の現在地**: この1年間で「純資産」がどれだけ増えたか、年間の平均貯蓄率を算出。
4. **5年後の未来予測**: 今のペースを維持した場合と、10%改善した場合の5年後の資産推移を比較。
5. **グランド・デザイン**: 来年に向けた「人生の支出」の優先順位を再定義して。
